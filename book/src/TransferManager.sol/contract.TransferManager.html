TransferManager

Inherits: Context, ERC2771Context

Author: Maurizio Murru https://github.com/akerbabber

A smart contract that manages token transfers and permits. This contract is used to limit the number of uses per day for each user. It also checks if the user is whitelisted.

This contract uses the ERC2771Context contract to handle meta transactions. And enables the use of USDC permit function in conjunction with meta-txs.
