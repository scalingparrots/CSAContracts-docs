CSASwap

Inherits: Initializable, AccessControlUpgradeable, PausableUpgradeable, UUPSUpgradeable, ERC2771ContextUpgradeable, CSASwapStorageContract

Author: Maurizio Murru https://github.com/akerbabber

This contract is used to swap various tokens and the native token for CSAT tokens. It uses Chainlink oracles to get the prices of tokens and supports pausing functionality.

Constructor for the CSASwap contract.

All variables are immutable, as such their values are hardcoded into the contract code
